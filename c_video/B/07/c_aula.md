## Aritiméticos:

 * [-] -> subtração
 * [*] -> multiplicação
 * [/] -> Divisão
 * [%] -> Resto
 * [**] -> Potencia

 *cuidado com a ordem dos fatores!* 
  Ordem de precedência: 
   ```
   () => ** => * => * / % => + -
   ```
