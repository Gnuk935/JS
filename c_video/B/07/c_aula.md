## Aritiméticos:
 * [+] -> adição
 * [-] -> subtração
 * [*] -> multiplicação
 * [/] -> Divisão
 * [%] -> Resto
 * [**] -> Potencia

 *cuidado com a ordem dos fatores!* 
  Ordem de precedência: 
   ```
   () => ** => * => * / % => + -
   ```
## Atribuição:
 * [=] -> recebe
 * [+=] -> adição
 * [-=] -> subtração
 * [*=] -> multiplicação
 * [/=] -> divisão
 * [%=] -> resto
 * [**=] -> exponencial
 * [<<=] -> bit-a-bit por deslocamento á esquerda
 * [>>=] -> bit-a-bit por deslocamento á direita
 * [>>>=]bit-a-bit deslocamento á direita não assinado
 * [&=] AND bit-a-bit
 * [^=]tribuição XOR bit-a-bit
 * [|=] OR bit-a-bit
