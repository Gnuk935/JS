## JS - B - 05 - Var e T. Primitivos
>Um resumo da aula cinco
## Variaveis:
* Armazena dados
* Tem um tipo
* Apenas um por vez
```` 
var [nome] = [valor] ;
````
> * *= -> recebe*
> * *" " ` -> formas de delimitar a string*
### Regras:
* Deve começar com:
    * Letra
    * $
    * _
* Não pode ter:
    * Numero como primeiro caracter
    * Espaços
    * Usar palavras reservadas
* Seguindo as outras:
    * Pd usar letras, numero e simbolos
    * Maiúsculas e min !=
    * Use nomes coerentes
------
## Tipos primitivos
* Os primordiais:
    * Number
    * String
    * Boolean
    * null
    * undefined
    * object
    * function
* Comando para saber o tipo:
````
typeof [valor]
````
