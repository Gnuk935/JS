## JS - D - 11 - Condições -- Part 1

### Sequências
 1. Comando
 2. Comando
 3. ...
 <br>
 São comandos que devem ser executados nessa ordem apenas!

### Quebrando a seq:
1. Comando <br>
**se algo acontecer: 2**
2. Comando: true <br>
**Se não acontecer: 3**
3. Comand: false <br>
**dps continua normalmente**
4. Comando
5. Comando 

### Usando:
* se: 
    ```` 
    if(condição){
        true
    }else{
        false
    }
    ````
